# ChestCavity 容器：`soul/`

本目录承载分魂系统的核心实现，包括：

- `fakeplayer/`：灵魂分身 `SoulPlayer` 的生命周期、指令桥接、工厂注册与事件监听。
- `navigation/`：分魂的寻路镜像、Baritone 集成以及测试支撑工具。
- `command/`、`runtime/` 等子模块：暴露调试指令、运行时处理器以及与容器/属性相关的工具。
