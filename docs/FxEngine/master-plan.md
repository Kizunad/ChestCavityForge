# FxEngine 时间线系统 · 总任务（Master Plan）

## 目标

- 提供可持续 FX 时间线能力，支持预算控制（默认关闭）与限流合并，服务 Shockfield 等模块。

## 里程碑

- M1：引擎骨架完成（FxTrack/FxTimelineEngine/FxScheduler），能调度简单轨道。
- M2：Builder/合并/预算控制落地；配置开关与阈值可调。
- M3：工具与示例编写；Shockfield 集成演示。
- M4：清理与测试；文档完善与验收。

## 关键交付物

- 文档：requirements / tech-framework / constraints / standards / implementation / tests
- 代码：引擎与门面、配置挂接、示例接线点（不含大规模粒子实现）
